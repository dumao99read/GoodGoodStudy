# 猜数字游戏（Bulls and Cows / 1A2B）🎮

一个经典的数字逻辑推理游戏，比简单猜数字更有挑战性。

## 🎯 游戏规则

### 基本规则
1. **目标数字**：从0-9中随机选取X个不重复的数字作为目标
2. **猜测机会**：玩家有Y次机会猜中目标数字
3. **结果反馈**：每次猜测后，系统返回A和B的结果：
   - **A**：数字和位置都正确
   - **B**：数字正确但位置错误
   - A的优先级比B高（已经记作A的数字不会再统计B）

### 示例说明
假设目标数字是 **1234**：
- 猜测 **1356** → **1A1B**
  - 1：数字和位置都正确 → 1A
  - 3：数字正确但位置错误 → 1B
- 猜测 **4321** → **0A4B**
  - 所有数字都正确，但位置全错 → 4B
- 猜测 **1234** → **4A0B**
  - 完全正确，游戏胜利！

## 🎮 游戏功能

- **开始新游戏** - 开始一轮新的猜数字游戏

- **游戏说明** - 查看详细的游戏规则说明
- **自定义设置** - 修改数字个数和尝试次数
- **难度预设** - 快速选择预设难度
- **分析工具** - 分析已猜测的数字和位置
- **查看历史** - 查看当前游戏的猜测历史
- **退出游戏** - 退出程序

### 2. 难度预设
- **简单模式**：3位数字，12次尝试
- **标准模式**：4位数字，10次尝试（默认）
- **困难模式**：5位数字，8次尝试
- **专家模式**：6位数字，6次尝试
- **大师模式**：7位数字，5次尝试

### 3. 智能提示系统
根据猜测结果给出不同的提示：
- **完全正确**：🎉 完美！完全正确！
- **差一个位置**：🔥 就差一个数字的位置了！
- **所有数字对位置错**：🔀 所有数字都对了，但位置需要调整！
- **大部分数字对**：👌 大部分数字都对了！
- **完全没有猜中**：💀 完全没有猜中，换个思路试试！

### 4. 分析工具
- 显示已尝试的数字
- 显示未尝试的数字
- 基于最后一次猜测的位置分析

## 🔍 游戏策略技巧

### 初级策略
1. **首次猜测**：尝试如0123、4567等连续数字
2. **分析反馈**：根据A和B的数量调整策略
3. **排除数字**：使用分析工具查看已尝试和未尝试的数字

### 高级策略
1. **信息最大化**：每次猜测尽可能获得最多信息
2. **位置推理**：根据B的数量推断数字位置
3. **系统排除**：使用排除法逐步缩小可能范围

## 🏆 胜利条件

- 在规定的尝试次数内完成

## 🔧 技术细节

### 依赖要求
- Python 3.6+
- 仅使用标准库（random, time, typing）

### 算法实现
1. **秘密数字生成**：使用random.shuffle确保不重复
2. **结果计算**：先计算A，再计算B（排除已算作A的数字）
3. **输入验证**：检查长度、数字、重复、范围
4. **历史记录**：保存所有猜测和结果供分析

### 错误处理
- 无效输入处理
- 键盘中断处理
- 异常捕获和友好提示
