# 猜数字游戏 🎮

一个简单但有趣的Python猜数字小游戏。

## 功能特点

- 🎯 随机生成1-100之间的数字
- 🔢 最多10次猜测机会
- 💡 智能提示系统（大小提示 + 接近程度提示）
- 📊 游戏统计记录
- ⚙️ 可自定义游戏设置
- 🏆 最佳成绩记录

## 如何运行

### 方法1：直接运行
```bash
python guess_number_game.py
```

### 方法2：使用Python解释器
```bash
python3 guess_number_game.py
```

### 方法3：在IDE中运行
1. 用PyCharm/VSCode等IDE打开文件
2. 点击运行按钮或按F5

## 游戏说明

### 主菜单选项
1. **开始新游戏** - 开始一轮新的猜数字游戏
2. **查看统计** - 查看游戏次数、最佳成绩等统计信息
3. **游戏说明** - 查看详细的游戏规则说明
4. **修改设置** - 修改数字范围、尝试次数等游戏设置
5. **退出游戏** - 退出程序

### 游戏规则
1. 程序会随机生成一个1-100之间的数字
2. 你有10次机会猜中这个数字
3. 每次猜测后，程序会告诉你：
   - 猜大了还是猜小了
   - 距离目标有多接近
4. 输入 'q' 可以随时退出当前游戏
5. 目标是尽可能用最少的次数猜中数字

### 提示系统
根据你猜测的数字与目标数字的差距，程序会给出不同的提示：
- **相差 ≤ 5**: "🔥 非常接近了！就差一点点！"
- **相差 ≤ 10**: "👍 很接近了！继续加油！"
- **相差 ≤ 20**: "👌 有点接近了，再调整一下。"
- **相差 ≤ 30**: "🤔 距离目标还有一段距离。"
- **相差 > 30**: "😅 离目标还比较远，大胆猜测！"

## 自定义设置

你可以通过菜单修改以下设置：
- **数字范围**: 可以设置为任意范围（如1-1000）
- **尝试次数**: 可以增加或减少最大尝试次数
- 设置会保存在当前游戏会话中
